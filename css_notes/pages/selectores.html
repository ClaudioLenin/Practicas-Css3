SELECTORS

TYPE HTML CSS
IDs <h1 id="main-title"> Our header </h1> #main-title {
color:red;
}
ATTRIBUTES <button disabled> Click </button> [disabled] {
color:red;
}
ELEMENTS <h1> Our header </h1>
<p> The Blog Post </p>
<div> More Info </div> 		h1 {
color:red;
}
CLASSES <h1 class="blog-post"> Our header </h1>
<p class="blog-post"> The Blog Post </p>
<div class="blog-post"> More Info </div> 		.blog-post {
color:red;
}
UNIVERSAL <h1> Our header </h1>
<p> The Blog Post </p>
<div> More Info </div> 		* {
color:red;
}

Notes:

  • "*" has the lowest priority
  • <Tag> and ::Pseudo elements have the second lowest priority
  • Class, :pseudo class and [attribute] selectors have the third lowest priority
  • Next in line are the #ID selector
  • Finally the inline styles will have highest priority
